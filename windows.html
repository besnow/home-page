如何使用

首先，找到文件夹中的猫咪 for windows.exe文件，双击运行。



打开猫咪 for Windows，已经是汉化成中文版的猫咪 for Windows。接下来就是如何使用中文版的猫咪 for Windows。




添加订阅

登录“贝雪云”账户，点击“我的订阅”——“一键订阅”——“复制订阅地址”


打开 猫咪 for Windows 应用程序，在左侧的标签页中选择「配置」， 在顶部输入您的订阅链接 ，然后点击「下载」按钮。


猫咪 for Windows 会自动拉取配置文件进行更新，如果一切顺利，你应当可以看到绿色提示信息「成功」，并且可以看到一个新增的配置文件：




启动 猫咪 for Windows

请务必确保开关是启用状态。 经博客聊天窗口统计，大量新手用户因未打开开关，导致连不上代理。

返回到「常规」部分，将「系统代理」的开关更改为绿色状态即可开始使用。此外，建议将「开机启动」也更改为绿色来让 猫咪 for Windows 在开机时自动启动。


现在，基本配置已经完成了，猫咪 for Windows已经开始工作了。



猫咪 for Windows的代理工作方式，分为全局，规则，直连，脚本四种。
