常見問題及解答

数据传输

什么是流量配额？

+ −

流量配额是指 BesnowCloud貝雪雲 订阅分配的最大数据传输量（流量），上传及下载流量均会计算在内，当流量配额用尽后，您将无法使用 BesnowCloud貝雪雲 订阅提供的服务。

流量配额每月会被重置吗？

+ −

如果没有特殊声明，所有 BesnowCloud貝雪雲 流量配额均会在每月自动重置。

BesnowCloud貝雪雲 订阅流量配额的重置日期不是每月 1 日，而是您在 BesnowCloud貝雪雲 购买订阅的日期。例如您在本月 10 日购买的订阅，那么您的订阅下次流量配额重置日期是下月 10 日。

如果您在已购订阅未到期的情况下更换了订阅，如无特殊声明，那么您的新订阅的流量配额重置日期会变为下月同日，且已使用流量无法立即重置，您需要等下月同日才能自动重置。例如您在本月 20 日更换了新订阅，那么您的新订阅下次流量重置日期为下月 20 日。

您可以在 BesnowCloud貝雪雲 管理面板中查看当前订阅距离下一次流量配额重置还要多长时间。

我可以为当月增加流量包配额吗？

+ −

不可以，BesnowCloud貝雪雲 当前不支持弹性流量包。如果您超出了订阅的流量配额，您只能等待流量配额自动重置，或选择付费重置流量配额。

我可以手动重置流量配额吗？

+ −

可以，您可以转至 BesnowCloud貝雪雲 管理面板中的「我的订阅」，在「操作」下拉菜单中，选择「重置当月流量」付费重置流量，您已使用的流量将被完全清空。如无特殊声明，流量费用将按照订阅月费用的 9 折收取。不限重置次数。

我可以通过续费提前重置流量配额吗？

+ −

不可以。续费购买的是订阅使用期限，不是每月的流量配额。

流量配额可以结转到下月吗？

+ −

不可以。我们不设置流量结转功能。

流量倍率是什么？

+ −

流量倍率是指一个数据包经过 BesnowCloud貝雪雲 网络时将被计算的比率。

例如，您使用一个 0.5 倍率的接入点，那么您使用该接入点消耗了实际 1GiB 流量时，那么在 BesnowCloud貝雪雲 中将会被计算为 512MiB。同理，您使用一个 10 倍率的接入点，那么您使用该接入点消耗了实际 1GiB 流量时，那么在 BesnowCloud貝雪雲 中将会被计算为 10GiB。

如果您的流量配额为 30GiB，您只用 0.5 倍率的接入点，那么理论上每月您最多可以通过 BesnowCloud貝雪雲 网络传输 60GiB 数据。同理，如果您只使用 10 倍率的接入点，那么理论上每月您最多可以通过 BesnowCloud貝雪雲 网络传输 3 GiB 数据。当然，您使用倍率为 0 的接入点传输任何数据都不会被计费。

BesnowCloud貝雪雲 流量倍率的制定是根据接入点服务器总流量配额、带宽及运营成本、业务承受能力等综合评估的，流量倍率的高低并不代表该接入点速率的快慢和好坏。

请注意，BesnowCloud貝雪雲 有权在不通知您的情况下，随时修改一个接入点的流量倍率。

我可以在流量配额完全用尽后继续使用 0 倍率接入点吗？

+ −

不可以，当您的 BesnowCloud貝雪雲 订阅流量配额完全用尽后，订阅中的所有接入点都将不再向您提供服务，并且短时间内不会有0倍率接入点。

访问中国境内的资源会不会计算流量？

+ −

BesnowCloud貝雪雲 并不设置分流功能。无论中国境内还是境外，只要数据包经过 BesnowCloud貝雪雲 接入点，都会消耗 BesnowCloud貝雪雲 的计算能力和带宽资源，所以都会被计算流量。

会提供流量明细吗？

+ −

BesnowCloud貝雪雲 会提供详细的流量使用记录。您可以在 BesnowCloud貝雪雲 管理面板的「流量明细」中查看您具体的流量明细。

购买订阅

我该如何购买订阅？

+ −

您需要先在 BesnowCloud貝雪雲 上注册一个账户，然后直接购买新订阅即可。您可以通过微信向我们付款。对于国际客户，您还可以联系客服，通过国际信用卡（MasterCard 和 Visa）付款。收款方可能会向您收取一定的手续费用。

每个人可以购买多少个订阅？

+ −

每个账户限制只能保有 1 个订阅。如果您需要开通多个订阅，您可以注册多个账户进行购买。

我可以随时变更订阅吗？

+ −

是的，您可以随时变更订阅，剩余订阅时长会按比例退款至您的余额账户中。为防止滥用，已使用流量恕不能被清空，且已使用流量需要在下月同日才能被重置。

我可以将订阅分享给他人使用吗？

+ −

不可以。基于防滥用原则，您购买的订阅默认仅允许您自己使用，请留意订阅介绍中规定的使用人数。无论哪种订阅，您都不可以将订阅合租、转借、泄露、二次销售给他人。

我可以在多个设备上使用吗？

+ −

您理论可以在无限台设备上使用，但这些设备同时连接的公网 IP 数量不可以超过套餐规定的数量，且应当位于同一地区。考虑到您可能需要在外出、通勤、工作时使用订阅，或在移动设备上通过移动网络使用订阅，我们允许您短时间超出此限制，但不能违反 BesnowCloud貝雪雲 条款和条件的公平使用原则。

公网 IP 不得长期超过合理地数量限制（订阅规定使用人数×3）。

我可以使用订阅进行 BT/PT 上传/下载吗？

+ −

不可以。我们无法判断 BitTorrent 协议传输的内容是否违反资源的最终用户许可协议(EULA)或者数字千年版权法(DMCA)，所以我们严禁在 BesnowCloud貝雪雲 网络上使用 BitTorrent 协议。

有部分软件及游戏启动器的下载加速也使用了 BitTorrent 协议，请在使用这些可能使用 BitTorrent 协议的软件之前，停止使用 BesnowCloud貝雪雲 订阅，以免被误判。

此外，BitTorrent 会严重占用服务器资源，这也违反了我们的公平使用原则。

我可以使用订阅玩游戏吗？

+ −

当然可以。我们并不限制您使用 BesnowCloud貝雪雲 订阅加速游戏，但我们的服务并未对游戏做了优化，这可能无法让您得到最佳体验，包括在高峰期可能的高延迟、延迟抖动或丢包。这一般不影响网页浏览、访问流媒体、下载，但对实时性要求较高的游戏可能有较大影响。

我可以在自动化程序上使用订阅吗？

+ −

不可以。我们禁止将订阅用于任何自动化程序，例如爬虫、机器人程序、人工智能、自动交易等。这违反了我们的公平使用原则。

BesnowCloud貝雪雲 订阅仅限人类使用。

我可以浏览非法网站或在网上发表自己的政治意见吗？

+ −

您在使用 BesnowCloud貝雪雲 订阅访问网站或某些资源时，不得违反接入点所在国家或地区的法律法规。您不可以使用 BesnowCloud貝雪雲 访问某些包含宣传极端政治、邪教、恐怖主义、儿童色情或某些内容上在大多数国家中属于犯罪行为的网站或资源。

除此之外，我们虽然尽可能地保障您自由的权利，但由于我们的部分工作人员仍然定居在中国境内，您在国际互联网上使用我们的服务发表自己对中国的政治意见时，可能会让我们的部分境内工作人员陷入麻烦，或导致部分境内接入服务器被迫关停影响其他用户使用，所以我们不允许您使用我们的服务发表影响中国的政治意见。

使用订阅

什么是订阅？

+ −

BesnowCloud貝雪雲 API 订阅是 BesnowCloud貝雪雲 提供的一套可在多个客户端上使用的通用应用程序接口。

API 支持多种客户端以及一些客户端的自动配置协议，这意味着您可以通过仅仅点击一次或几个简单步骤来快速完成客户端的配置而无需手动设置客户端并填写配置文件。

如何使用订阅？

+ −

请先注册登录然后可以参照我们的 “使用文档” ，了解如何在受支持的客户端上使用订阅。

我不小心泄露了订阅的 URL 链接，该怎么办？

+ −

请转至 BesnowCloud貝雪雲 管理面板中的「我的订阅」，点击「操作」下拉菜单中的「重置订阅信息」按钮，重置您的订阅。

请务必妥善保管好您的订阅信息。无论是您主动分享订阅信息亦或者被他人盗用，造成的后果均由您自行承担。

账户

我忘记了我的账户或密码，该怎么办？

+ −

如果您忘记了您的密码，请点击 BesnowCloud貝雪雲 管理面板登录页下方的「忘记密码」选项，然后输入您在 BesnowCloud貝雪雲 上的注册邮箱，然后点击「发送」按钮，系统会发送一封验证码给您，然后您便可重置您的密码。

如果您遗忘了您在 BesnowCloud貝雪雲 上的注册邮箱，请加入我们的 Telegram 群组，联系工作人员处理。

我该如何删除我的账户？

+ −

目前尚不支持您自主删除您的账户。如果您决定不再使用我们的服务，请联系我们的工作人员删除您的账户。

请注意您在删除账户后，您在 BesnowCloud貝雪雲 上的任何资料都将被完全删除，且无法恢复，包括已开通的订阅或尚未用完的账户余额等，请再三确认后再向我们要求删除账户。

我的账户余额可以用来干什么？

+ −

您的账户余额可以用于支付订阅续期费用、重置流量、购买新订阅，但账户余额无法用于提现，包括但不限于 Paypal、支付宝、微信等。

我可以申请退款吗？

+ −

基于防滥用原则，我们尚不支持退款。但您可以要求我们将您当前的剩余订阅按比例退款至余额账户中。
